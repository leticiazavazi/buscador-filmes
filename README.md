# 🎬 Buscador de Filmes

Aplicação web desenvolvida com React JS que permite buscar filmes pelo nome utilizando a API gratuita do OMDb.

## 🚀 Como usar

1. Clone o repositório:
```bash
   git clone https://github.com/SEU_USUARIO/buscador-filmes.git
```

2. Instale as dependências:
```bash
   npm install
```

3. Inicie o servidor de desenvolvimento:
```bash
   npm run dev
```

4. Acesse `http://localhost:5173` no navegador.

## 📦 Bibliotecas utilizadas

- [React](https://react.dev/) — biblioteca principal para construção da interface
- [Vite](https://vitejs.dev/) — ferramenta de build e servidor de desenvolvimento

## 🌐 API utilizada

- [OMDb API](https://www.omdbapi.com/) — API gratuita de filmes (requer cadastro para obter chave)
